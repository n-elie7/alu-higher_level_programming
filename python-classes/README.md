# Python - Classes and Objects

This project contains exercises focused on understanding and implementing classes and objects in Python.

## Description

Learning objectives include:
- Understanding Object-Oriented Programming (OOP)
- Creating and using classes
- Working with attributes and methods
- Understanding encapsulation and data abstraction

## Environment

- **OS**: Ubuntu 20.04 LTS
- **Python Version**: 3.8.5
- **Style Guide**: pycodestyle 2.7.*

## Requirements

- All files must be executable
- All files must end with a new line
- First line of all files: `#!/usr/bin/python3`
- All modules, classes, and functions must have documentation
- Code must follow pycodestyle standards

## Files

- `0-square.py` - Empty class Square that defines a square
