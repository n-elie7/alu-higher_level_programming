Test cases for MyList class
============================

This module contains doctests for the MyList class.

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Test basic functionality:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Test with single element:
    >>> single = MyList()
    >>> single.append(1)
    >>> single.print_sorted()
    [1]

Test with negative numbers:
    >>> negative_list = MyList()
    >>> negative_list.append(-5)
    >>> negative_list.append(3)
    >>> negative_list.append(-1)
    >>> negative_list.append(0)
    >>> negative_list.print_sorted()
    [-5, -1, 0, 3]

Test with already sorted list:
    >>> sorted_list = MyList()
    >>> sorted_list.append(1)
    >>> sorted_list.append(2)
    >>> sorted_list.append(3)
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Test with reverse sorted list:
    >>> reverse_list = MyList()
    >>> reverse_list.append(5)
    >>> reverse_list.append(4)
    >>> reverse_list.append(3)
    >>> reverse_list.append(2)
    >>> reverse_list.append(1)
    >>> reverse_list.print_sorted()
    [1, 2, 3, 4, 5]

Test with duplicate values:
    >>> dup_list = MyList()
    >>> dup_list.append(3)
    >>> dup_list.append(1)
    >>> dup_list.append(3)
    >>> dup_list.append(2)
    >>> dup_list.append(1)
    >>> dup_list.print_sorted()
    [1, 1, 2, 3, 3]

Test inheritance from list:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Test that MyList inherits list methods:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> len(my_list)
    3
    >>> my_list[0]
    3
    >>> my_list.pop()
    2
    >>> print(my_list)
    [3, 1]

Test with large numbers:
    >>> large_list = MyList()
    >>> large_list.append(1000000)
    >>> large_list.append(1)
    >>> large_list.append(500000)
    >>> large_list.print_sorted()
    [1, 500000, 1000000]

Test initialization with values:
    >>> init_list = MyList([3, 1, 2])
    >>> init_list.print_sorted()
    [1, 2, 3]
